# 游戏元素缩放配置说明

## 📍 配置位置
文件：`src/components/PixiGame.js`  
位置：文件顶部，第 14-60 行

## 🎮 配置对象：ELEMENT_SCALES

所有游戏元素的大小都在这个对象中统一配置，修改这里的数值就能调整元素大小。

### 📦 配置结构

```javascript
const ELEMENT_SCALES = {
  // 瓶子相关
  bottle: {
    inSea: 1.0,           // 海里漂浮的瓶子 (bottle1.png)
    throwing: 1.2,        // 扔出时的瓶子 (bottle.png)
    landed: 1.2,          // 落地后的瓶子 (bottle1.png)
    caught: 0.03,         // 被捕获的瓶子
  },
  
  // 场景元素
  island: {
    left: 0.8,            // 左侧小岛
    right: 1.5,           // 右侧小岛
  },
  
  title: 0.8,             // 标题
  
  // 动物和自然元素
  fish: {
    min: 5,               // 小鱼最小尺寸
    max: 15,              // 小鱼最大尺寸
  },
  
  bubble: {
    min: 2,               // 气泡最小尺寸
    max: 10,              // 气泡最大尺寸
  },
  
  cloud: {
    min: 0.5,             // 云朵最小缩放
    max: 1.5,             // 云朵最大缩放
  },
  
  // 太阳
  sun: {
    outerGlow: 60,        // 外部光晕半径
    middleGlow: 45,       // 中间光晕半径
    innerGlow: 35,        // 内部光晕半径
    body: 30,             // 太阳主体半径
    innerSun: 15,         // 内部纹理半径
  },
  
  // 工具
  net: 0.4,               // 渔网
};
```

## 🔧 如何修改

### 1. 修改瓶子大小
想让所有瓶子都变大一点：
```javascript
bottle: {
  inSea: 1.5,      // 从 1.0 改为 1.5
  throwing: 1.5,   // 从 1.2 改为 1.5
  landed: 1.5,     // 从 1.2 改为 1.5
  caught: 0.05,    // 从 0.03 改为 0.05
}
```

### 2. 修改小岛大小
让左边小岛更大：
```javascript
island: {
  left: 1.2,       // 从 0.8 改为 1.2
  right: 1.5,      // 保持不变
}
```

### 3. 修改小鱼大小范围
让小鱼整体变大：
```javascript
fish: {
  min: 10,         // 从 5 改为 10
  max: 25,         // 从 15 改为 25
}
```

### 4. 修改太阳大小
让太阳整体变小：
```javascript
sun: {
  outerGlow: 40,   // 从 60 改为 40
  middleGlow: 30,  // 从 45 改为 30
  innerGlow: 23,   // 从 35 改为 23
  body: 20,        // 从 30 改为 20
  innerSun: 10,    // 从 15 改为 10
}
```

## 📊 数值说明

### 缩放倍数类型
- **固定倍数**（如 `title: 0.8`）：元素会按这个倍数缩放
- **范围值**（如 `fish: { min: 5, max: 15 }`）：元素会在这个范围内随机生成

### 实际大小计算
所有数值都会乘以 `scaleFactor`（响应式缩放因子）：
```
实际大小 = 配置值 × scaleFactor
```

例如在 750px 宽度屏幕上：
- `scaleFactor = 1.0`
- `bottle.inSea = 1.0`
- 实际瓶子大小 = `1.0 × 1.0 = 1.0`

在 375px 宽度屏幕上：
- `scaleFactor = 0.5`
- `bottle.inSea = 1.0`
- 实际瓶子大小 = `1.0 × 0.5 = 0.5`

## ⚠️ 注意事项

1. **保持比例协调**：修改一个元素时，注意与其他元素的大小关系
2. **测试不同屏幕**：修改后在不同屏幕尺寸下测试效果
3. **范围值要合理**：`min` 必须小于 `max`
4. **不要设置为 0**：会导致元素不可见
5. **太大会超出屏幕**：建议在 0.1 - 2.0 之间调整

## 🎯 常见调整场景

### 场景1：所有瓶子统一大小
```javascript
bottle: {
  inSea: 1.0,
  throwing: 1.0,
  landed: 1.0,
  caught: 0.03,
}
```

### 场景2：让游戏元素整体变小
所有数值乘以 0.8：
```javascript
bottle: { inSea: 0.8, throwing: 0.96, landed: 0.96, caught: 0.024 }
island: { left: 0.64, right: 1.2 }
title: 0.64
// ... 其他元素类似
```

### 场景3：只调整装饰元素
```javascript
fish: { min: 3, max: 8 }      // 小鱼变小
bubble: { min: 1, max: 5 }    // 气泡变小
cloud: { min: 0.3, max: 1.0 } // 云朵变小
```
