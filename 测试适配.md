# 测试移动端适配

## 重启服务器

1. 停止当前运行的服务器 (Ctrl+C)
2. 重新运行:
```bash
npm run dev
```

## 如何验证适配是否生效

### 方法1: 检查浏览器开发者工具

1. 打开浏览器开发者工具 (F12)
2. 切换到 Elements/元素 标签
3. 查看任意元素的样式
4. 如果看到 `width: 7.29167vw` 这样的单位,说明适配已生效
5. 原来的 `width: 140px` 会被转换为 `width: 7.29167vw`

### 方法2: 调整浏览器窗口大小

1. 拖动浏览器窗口改变大小
2. 观察UI元素是否自动缩放
3. 按钮、图片应该随窗口大小等比例变化

### 方法3: 使用移动设备模拟

1. F12 打开开发者工具
2. 点击设备工具栏图标 (或按 Ctrl+Shift+M)
3. 选择不同的设备 (iPhone, iPad等)
4. 观察UI是否完美适配

## 预期效果

- ✅ 所有按钮大小自动适配
- ✅ 图片尺寸自动缩放
- ✅ 间距和边距保持比例
- ✅ 不同屏幕尺寸显示一致

## 如果还是没生效

检查以下几点:

1. **清除缓存**: Ctrl+Shift+R 强制刷新
2. **检查控制台**: 看是否有错误信息
3. **确认依赖**: 运行 `npm list postcss-px-to-viewport`
4. **重新安装**: 
   ```bash
   rm -rf node_modules
   npm install
   npm run dev
   ```

## 注意事项

- postcss只处理CSS中的px
- PixiJS游戏场景仍使用JS缩放
- 媒体查询中的px不会被转换(可配置)
